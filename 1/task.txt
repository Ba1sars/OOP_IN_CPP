Вариант № 18: «Генератор Структур»
Реализовать функцию для генерации исходного кода структуры на основе её представления
в формате json.
Предусмотреть следующие типы данных:
• строка — char* (поле состоит из произвольных символов внутри кавычек)
• целое число — long (поле состоит из цифр без кавычек)
• дробное число — double (поле состоит цифр и символа «.» без кавычек)
• логический тип — bool (поле состоит из true или false без кавычек)
• неизвестный тип — void* (поле состоит из null)
На вход функции передаётся строка с описанием экземпляра структуры в формате json и
название структуры. В качестве результата функция должна вернуть исходный код
структуры. В случае, если json некорректен или присутствует любой неподдерживаемый тип
(массив или объект), вернуть ошибку.
Экранирование символов во входной строке допускается не поддерживать.
Например, в качестве аргументов могут передаваться название структуры «MyStruct» и json
строка:
{
"a": "123",
"b": 123,
"c": 123.4,
"d": false,
"e": null
}
В этом случае в качестве результата необходимо вернуть строку:
struct MyStruct {
char* a;
long b;
double c;
bool d;
void* e;
};
Реализовать 2 перегрузки описанной функции:
1. Принимает на вход и возвращает строки в виде указателя (const char*).
2. Принимает на вход и возвращает строки в виде экземпляра std::string.
* Тип возвращаемого из каждой перегрузки значения должен совпадать по стилю с типом
передаваемого.Для всех типов объявить using через класс со статическим членом.Если передаётся тип данных из языка C++ (string, vector, …), то и возвращать
необходимо тип из C++. Если передаётся сырой тип из C (char*, int*, …), то вернуть также
нужно тип из C.
