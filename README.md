# Описание дирректив
## Папка 1:
- В данной лабораторной содержиться простая программа, которая трансформирует вводимые данные в JSON формат. *Вариант лабораторной находится в файле task.txt*.
## Папка 2:
- В данной лабораторной была произведена работа с классами в C++, а так же была работа с динамической памятью в С++ с помощью **new/delete**. *Вариант лабораторной работы находится в файле task.txt*.
## Папка 2ex:
- В данной лабораторной нужно было взять **Лабораторную Работу номер 2** другого студента и написать main-функцию, использующую библиотеки из варианта этого студента. Также нужно было проверить код студента на правильность и по возможности исправить ошибки, которые он совершил в результате выполнения лабораторной. 
## Папка 3:
- В данной лабораторной надо было использовать накопленный опыт, чтобы разработать компьютерную игру со всеми аспектами, например, графика, логика врагов (ИИ), и тд. Главное в этом варианте - это соблюдать принципы SOLID. Также надо было разработать диаграмму классов перед тем, как писать код. Ещё одним обязательным условием была разработка собственного контейнерного класса, то есть со всеми иттераторами и контейнерами, соответсвующим критериям компилятора.
